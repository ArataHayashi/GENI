SUMMARY

GENI is an open web tool for rapid analysis of Gene Set Enrichment which enables intuitive analysis of gene set enrichment in cancer patients' gene expression without prior software experience.
It gives you inspiration on how the gene of your interest affects a set of genes based on the gene expression of cancer patients from clinical studies.





Index

    1. Start GENI
     Result
    2. Apply GSEA on your own data
     Result
    3. Download gene expression data





1.Start GENI

1. Enter your gene of interest.
 Enter a gene symbol or gene ID (NCBI gene ID).
 The candidate genes will appear below, you can copy and paste it.
 Once you entered the gene name, it shows "Gene ID is valid".

2. Select study that you want apply GENI on.
     Next, select one of the study form the TCGA database.
     You can choose the tissue type first. Default is "all" to show all studies.
     Once you chose the study, it shows chosen study name below.

3. Select gene set that used to find group of genes.
     Then, select one of the gene sets form MSigDB.
     For more information about gene sets, click the link here (open in new tab).

4. Change detailed settings of the run (optionally).
     You can change detailed settings for the GENI.
       -Select first order of the sheet : You can define the order of the result summary sheet.
        You can change it later on the sheet itself,too.
       -Change permutations : You can define the number of permutations.
       -Change minimun number of gene set
       -Change maximun number of gene set
       -Select normalization option
       -Select exponent option
       -Change p-value cut off







Result


Once you click "Apply GSEA", you will intermediate result table with gene set name, gene set size, NES(normalized enrichment score), Nominal p-value, Family-wise error rate p-value and False discovery rate q-value.(See Example)


Below the table, you can find "summary plot" which colored by FDR, sized by gene set size and ordered by p-value.
You can hide / show the "summary plot" by clicking the "Show/Hide summary plot" button.


Now, You can chose single row to see its details, enrich plot and gene list. Selected row will be colored blue
Or, you can click first column of several rows to select gene group for download.
Selected gene groups are marked as ✅.


When you select single row, you can find details below the "summary plot".
After you chose row, you can click the "Click for gene set detail" button and find details about its gene sets at MSigDB website.
Optionally, you can select the "put p value table" to show details in the enrich plot. It also affect on the plots you download.
In the end, you can download your analysis result by clicking "download" button. Now window to chose the directory will be appear and the file contains summary plots, summary result, colleration sheet of the data, selected gene sets' enrich plot and selected gene sets' gene list will be downloaded.





2.Apply GSEA on your own data




As a sub-function of GENI, you can easily apply Gene Set Enrichment Analysis on your results.
First, click the "Browse" button and upload your data to the GENI with gene name or gene ID in first column and expression difference in second column in xlsx, csv or txt format.
Your uploaded file format will be appeared below.

Second, elect gene set that used to find group of genes.
     Then, select one of the gene sets form MSigDB.
     For more information about gene sets, click the link here (open in new tab).

Optionally, you can change detailed settings of the run.
     You can change detailed settings for the GENI.
       -Select first order of the sheet : You can define the order of the result summary sheet.
        You can change it later on the sheet itself,too.
       -Change permutations : You can define the number of permutations.
       -Change minimun number of gene set
       -Change maximun number of gene set
       -Select normalization option
       -Select exponent option
       -Change p-value cut off




Result

Below the table, you can find "summary plot" which colored by FDR, sized by gene set size and ordered by p-value.

Now, You can chose single row to see its details, enrich plot and gene list. Selected row will be colored blue
Or, you can click first column of several rows to select gene group for download.
Selected gene groups are marked as ✅.


When you select single row, you can find details below the "summary plot".
